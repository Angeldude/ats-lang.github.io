Chapter 17. Introduction to Function Templates

The support for function templates in ATS is deeply ingrained in the design and implementation of ATS. Primarily, function templates are meant to provide a general approach to code reuse in ATS that is highly flexible (in terms of applicability) while incurring minimal run-time overhead if any.

Function Templates for Constructing Loops

Beginners in functional programming who have already acquired some knowledge of imperative programming often ask about ways to construct for-loops and while-loops. A common answer is that loop constructs are unnecessary in functional programming as they can be readily replaced with higher-order functions. I would now like to point out some issues with this answer.
